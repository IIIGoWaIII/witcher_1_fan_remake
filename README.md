# Witcher 1 Fan Remake

A fan remake of the classic Witcher 1 game built using the **Godot 4** engine.

## Features

- **Engine:** Godot 4.6 (Forward Plus)
- **Physics Engine:** Jolt Physics
- **Renderer:** Direct3D 12 (Windows)
- **Camera System:** Dual camera modes (FPS and TPS) with seamless switching.
- **Player Movement:** Character body movement with support for WASD and jumping.
- **Assets:** High-quality character model for Geralt of Rivia.

## Controls

- **W/A/S/D:** Move character
- **Space:** Jump
- **C/V:** Switch camera (FPS <-> TPS)
- **Mouse:** Look/Rotate

## Development

This project is in its early stages. Currently, basic movement and camera systems are implemented.

### Prerequisites

- [Godot Engine 4.x](https://godotengine.org/)
- [Godot Jolt Physics Engine Plugin](https://github.com/godot-jolt/godot-jolt)
